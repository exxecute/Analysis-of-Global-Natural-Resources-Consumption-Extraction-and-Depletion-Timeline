# Ход работы по очистке и унификации энергетических данных

В ходе проекта была собрана исходная информация из двух основных источников: **Our World in Data (OWID)** и **BP Energy Institute**. Для унификации данных была разработана **структура таблицы**, включающая колонки:

```
country, iso_code, region, year, resource, metric, value, unit, source
```


Из набора OWID использовались показатели по потреблению и производству энергии, электроэнергии, возобновляемых источников и выбросам CO₂, а из BP — данные по запасам и добыче топлива, а также по производству энергии по видам ресурсов.

Далее была проведена **очистка и нормализация данных**: приведение названий стран к единому виду, фильтрация только используемых ресурсов (`Oil`, `Gas`, `Coal`, `Electricity`, `Energy`, `CO2`) и метрик (`Consumption`, `Production`, `Reserves`, `Emissions`), удаление агрегированных регионов и исторических стран, а также приведение года и значений к числовому формату. Для каждой страны были добавлены **ISO-код и регион** из справочника, что позволило унифицировать данные и обеспечить корректный merge.

В результате была получена **объединённая панельная таблица**, готовая для анализа: каждая строка соответствует стране, году, ресурсу и метрике, при этом значения представлены в стандартизированных единицах. Таблица сохраняется в формате CSV и полностью готова к дальнейшему анализу и построению визуализаций по потреблению и производству энергии, доле возобновляемых источников и выбросам CO₂.
